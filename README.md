# 🎶 Band Name Generator

Welcome to the **Band Name Generator**!  
This is a simple Python program that creates a fun band name based on your personal inputs.

---

## 🚀 How It Works

The program asks you two questions:

1. Which city did you grow up in?
2. What is the name of your pet?

It then combines your answers to generate a unique band name!

---

## 💻 Code

```python
print("Welcome to the band name generator.")

city = input("Which city did you grow up in?\n")
pet = input("What is the name of your pet?\n")

print(f"Your band name could be {city} {pet}.")
````

---

## ▶️ How to Run

1. Make sure you have Python installed.
2. Save the code in a file, for example:

   ```
   main.py
   ```
3. Run the program:

   ```
   main.py
   ```
4. Enter your answers and enjoy your new band name! 🎤
